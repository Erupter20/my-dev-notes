**Horizontal**

HTML:
```
<div class="card-container-column"> 
<div class="card">Card A</div> 
<div class="card">Card B</div>
<div class="card">Card C</div>
</div>
```

/* Basic Styling for the Cards */
```
.card {
  padding: 20px;
  border: 1px solid #ccc;
  border-radius: 8px;
  background-color: #f9f9f9;
  text-align: center;
}

/* 1. Horizontal (Row) Layout */
.card-container-row {
  display: flex;
  flex-direction: row; /* Default behavior, places items side-by-side */
  gap: 20px;
}

.card-container-row .card {
  flex: 1; /* This tells each card to grow and share the space equally */
}
```



/* 2. Vertical (Column) Layout */
```
.card-container-column {
  display: flex;
  flex-direction: column; /* This is the key change to stack items */
  gap: 20px;
  align-items: center; /* Optional: centers the stacked cards on the page */
}

.card-container-column .card {
  width: 80%; /* Give cards a specific width in a column layout */
}
```
